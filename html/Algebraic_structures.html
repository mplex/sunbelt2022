Elementary structures

An example of an elementary structure is found in Dihedral group.

Dihedral group of the Equilateral Triangle

The Dihedral group of the Equilateral Triangle, denoted as \(D_3\), is an example of group structure. In order to construct it, we start by finding all possible combinations of three elements, and follow the process.

pmd <- data.frame(matrix(NA, ncol=3, nrow=0))
for(i in rev(seq_len(nrow(pm)))) {
    pmv <- as.vector(unlist(pm[i,]))
# make sure that each element appears just once
    if(isTRUE(sum(pmv)==6L)==TRUE && length(unique(pmv))==ncol(pm)) { 
       pmd[nrow(pmd)+1L,] <- as.vector(pm[i,]) } 
    }
